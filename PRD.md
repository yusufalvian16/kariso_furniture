## 1. GAMBARAN UMUM PROYEK

Membangun sebuah **landing page** untuk Kariso Furniture yang berfungsi sebagai:

## 3. STRUKTUR HALAMAN (SECTIONS)

Website terdiri dari **7 seksi utama** yang disusun secara berurutan dari atas ke bawah:

---

### SEKSI 1 — HERO (Bagian Utama / First Impression)

**Tujuan:** Menarik perhatian pengunjung dalam detik pertama dan mendorong aksi kontak.

**Konten yang harus ada:**
- **Headline utama** (teks besar, tegas):  
  Contoh: *"Wujudkan Hunian Impian Anda Bersama Kariso Furniture"*
- **Sub-headline** (1–2 kalimat pendek):  
  Contoh: *"Kitchen set & desain interior kustom berkualitas, harga terjangkau, siap melayani seluruh wilayah Yogyakarta."*
- **Tombol CTA (Call to Action) utama:**  
  Label: **"Konsultasi Gratis via WhatsApp"**  
  Aksi: Link ke `https://wa.me/6281334478963`  
  Warna: mencolok (hijau WhatsApp atau warna aksen brand)
- **Background:** Area untuk gambar/foto produk berkualitas tinggi (kitchen set atau ruang interior — placeholder image jika belum ada aset)

**Catatan desain:**
- Teks headline harus terbaca dengan jelas di atas background gambar
- Tombol CTA harus terlihat tanpa perlu scroll

---

### SEKSI 2 — KEUNGGULAN / WHY CHOOSE US

**Tujuan:** Membangun kepercayaan dan membedakan Kariso dari kompetitor.

**Konten — tampilkan sebagai 4 kartu/ikon:**

| Ikon | Judul | Deskripsi Singkat |
|---|---|---|
| ✏️ / Pensil | **Desain 100% Kustom** | Setiap produk dirancang sesuai kebutuhan dan selera Anda — mulai dari dimensi, material, hingga finishing. |
| 💬 / Chat | **Konsultasi Gratis** | Tim ahli kami siap membantu Anda tanpa biaya di awal. Diskusikan visi Anda, kami hadirkan solusinya. |
| 💰 / Harga | **Harga Terjangkau, Kualitas Terjaga** | Kami menawarkan harga yang bersaing tanpa mengorbankan kualitas bahan dan kerapian pengerjaan. |
| 🚚 / Truk | **Melayani Seluruh Wilayah** | Dari Sleman, Bantul, Kulon Progo, Gunungkidul, hingga luar DIY — kami siap hadir ke lokasi Anda. |

### SEKSI 3 — LAYANAN KAMI

**Tujuan:** Menampilkan jenis layanan yang ditawarkan secara jelas.

**Konten — tampilkan sebagai daftar atau grid kartu:**

1. **Kitchen Set Kustom**  
   Dirancang khusus sesuai ukuran dan selera Anda. Tersedia dalam berbagai pilihan material dan finishing.

2. **Desain Interior**  
   Penataan ruang dalam hunian maupun komersial — mulai dari konsep hingga eksekusi.

3. **Dekorasi Eksterior**  
   Solusi tampilan luar bangunan yang estetis dan fungsional.

4. **Furnitur Kustom**  
   Berbagai jenis furnitur sesuai kebutuhan spesifik ruangan Anda.

**Catatan desain:** Setiap kartu layanan bisa menggunakan ikon dan placeholder gambar produk.

---

### SEKSI 4 — PORTOFOLIO / GALERI PROYEK

**Tujuan:** Memperlihatkan hasil kerja nyata untuk membangun kepercayaan visual.

**Konten:**
- Grid galeri foto (minimal 6–9 placeholder slot gambar)
- Kategori filter opsional: *Semua / Kitchen Set / Interior / Eksterior*
- Setiap gambar dapat diberi label singkat (contoh: *"Kitchen Set Minimalis Modern — Sleman"*)

**Spesifikasi material yang bisa ditampilkan sebagai label/badge opsional:**
- Material: Multiplek / Blockboard / Kayu Solid
- Finishing: HPL / Cat Duco / Melamik

**Catatan:** Gunakan placeholder image bertema interior/furnitur jika foto asli belum tersedia. Beri petunjuk komentar pada kode untuk memudahkan penggantian gambar.

---

### SEKSI 5 — ALUR KERJA / PROSES PENGERJAAN

**Tujuan:** Menjelaskan proses kerja secara transparan untuk mengurangi kekhawatiran calon pelanggan.

**Konten — tampilkan sebagai timeline horizontal atau vertikal (4 langkah):**

1. **Konsultasi Gratis**  
   Hubungi kami, ceritakan kebutuhan dan keinginan Anda. Tanpa biaya, tanpa komitmen.

2. **Survei & Pengukuran**  
   Tim kami datang ke lokasi untuk mengukur ruangan dan menyesuaikan desain dengan kondisi aktual.

3. **Desain & Persetujuan**  
   Kami buat desain 3D, pilih material, dan finalisasi bersama Anda sebelum produksi dimulai.

4. **Produksi & Instalasi**  
   Furnitur diproduksi di workshop kami dan dipasang langsung di lokasi Anda dengan standar presisi tinggi.

## 5. SPESIFIKASI TEKNIS

### 5.1 Tech Stack (Rekomendasi untuk AI Agent)
- **HTML + CSS + Vanilla JavaScript** (single file, tanpa framework berat)  
  *ATAU*
- **React / Next.js** jika agent mendukung

### 5.2 Responsivitas
- **Mobile-first design** — mayoritas pengunjung dari perangkat mobile
- Breakpoint: Mobile (< 768px), Tablet (768–1024px), Desktop (> 1024px)
- Gambar harus dioptimasi (lazy loading)

### 5.3 Performa
- Gambar menggunakan `loading="lazy"`
- Hindari library/font berat yang tidak perlu
- Target: loading cepat di koneksi mobile

### 5.4 SEO Dasar
- `<title>`: *Kariso Furniture – Kitchen Set & Desain Interior Kustom Yogyakarta*
- `<meta description>`: *Kariso Furniture menyediakan kitchen set kustom, desain interior dan eksterior berkualitas dengan harga terjangkau di Yogyakarta, Sleman, Bantul, dan sekitarnya.*
- Tag `<h1>` hanya satu per halaman (ada di Hero)
- Alt text pada semua gambar
- Schema markup `LocalBusiness` di `<head>`:
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "Kariso Furniture",
  "address": {
    "@type": "PostalAddress",
```

## 6. PANDUAN DESAIN VISUAL

### 6.1 Palet Warna (Rekomendasi)
| Peran | Warna | Hex |
|---|---|---|
| Warna Utama (Primary) | Cokelat kayu hangat / Tembaga | `#8B5E3C` atau `#6B4226` |
| Warna Aksen (CTA) | Hijau WhatsApp | `#25D366` |
| Background | Putih bersih / Krem lembut | `#FFFFFF` / `#FAF7F2` |
| Teks Utama | Abu-abu gelap | `#2D2D2D` |
| Teks Sekunder | Abu-abu medium | `#6B6B6B` |

> *Catatan: Warna bisa disesuaikan dengan branding aktual Kariso jika ada referensi.*

### 6.2 Tipografi (Rekomendasi)
- **Heading:** Google Fonts — `Playfair Display` (elegan, kesan premium) atau `Poppins Bold`
- **Body:** `Poppins Regular` atau `Inter`
- Ukuran minimum body: 16px (untuk keterbacaan mobile)

### 6.3 Gaya Visual Keseluruhan
- Bersih dan modern (clean & minimal)
- Kesan premium namun tetap hangat dan terjangkau
- Banyak whitespace antar seksi
- Gunakan ikon SVG sederhana (bukan emoji) untuk kartu keunggulan dan proses

## 8. CATATAN IMPLEMENTASI UNTUK AI AGENT

1. **Gunakan placeholder image** dari Unsplash atau LoremFlickr untuk kategori `furniture`, `kitchen`, `interior` selama aset asli belum ada
2. **Semua tombol CTA** harus mengarah ke WhatsApp: `https://wa.me/6281334478963`
3. **Jangan masukkan** informasi internal seperti detail rekrutmen, struktur gaji, atau data operasional internal bisnis
4. **Komentar kode** yang jelas untuk setiap seksi agar mudah diedit pemilik
5. **Smooth scroll** saat klik menu navigasi ke seksi terkait
6. **Animasi ringan** (fade-in saat scroll) boleh ditambahkan untuk kesan profesional, namun tidak boleh memperlambat loading
7. **Floating WhatsApp button** harus ada di semua ukuran layar
