# Rencana Pengembangan Website Kariso Furniture 🚀

Dokumen ini berisi ide dan strategi untuk meningkatkan performa, SEO, dan konversi website Kariso Furniture setelah peluncuran awal.

---

## 1. Strategi SEO & Visibilitas (Prioritas Utama)
Agar website muncul di halaman pertama Google saat orang mencari kata kunci seperti "Kitchen Set Jogja" atau "Interior Sleman".

- [ ] **Technical SEO:**
    - Buat file `sitemap.xml` agar Google mudah mengindeks semua bagian web.
    - Buat file `robots.txt` untuk memberi instruksi pada bot Google.
    - Pasang Google Search Console untuk memantau trafik dan kata kunci.
- [ ] **Local SEO:**
    - Optimalkan Google Business Profile (Google Maps) Kariso Furniture dan tautkan ke website ini.
- [ ] **Content Marketing (Artikel/Blog):**
    - Pindah dari "Single Section Tips" ke "Multi-page Blog".
    - Target Artikel Bulanan:
        - *Panduan Biaya Pembuatan Kitchen Set di Jogja 2025.*
        - *Perbandingan Material: Plywood vs Blockboard vs MDF.*
        - *7 Tren Warna Kitchen Set Minimalis Tahun Ini.*

---

## 2. Optimasi Aset & Visual
- [ ] **Foto Asli:** Ganti semua foto placeholder Unsplash dengan foto proyek asli Kariso Furniture (ini sangat krusial untuk kepercayaan pelanggan).
- [ ] **Video Walkthrough:** Tambahkan video pendek (misal dari TikTok/Reels) yang menunjukkan hasil pemasangan furniture di lokasi klien.
- [ ] **Optimasi Gambar:** Pastikan semua foto yang diupload sudah di-compress (format .webp) agar website tetap "wus-wus" (cepat).

---

## 3. Fitur Tambahan & Konversi
- [ ] **Katalog PDF:** Tambahkan tombol "Unduh Price List/Katalog" yang meminta nomor WA (Lead Generation).
- [ ] **Testimoni Video:** Jika ada, tambahkan video singkat pelanggan yang puas untuk meningkatkan kredibilitas di atas 100%.
- [ ] **FAQ Section:** Tambahkan tanya jawab umum (Lama pengerjaan, sistem DP, garansi, dll) untuk mengurangi beban pertanyaan berulang di WhatsApp.

---

## 4. Maintenance Teknis
- [ ] **Tailwind Build:** Selalu jalankan `npm run build` setelah mengubah class CSS sebelum melakukan push ke GitHub.
- [ ] **Update Versi:** Cek berkala update Tailwind CSS atau plugin lainnya untuk memastikan keamanan dan performa terbaik.
